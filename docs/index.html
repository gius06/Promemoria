Sistema di gestione delle attività per creare, modificare, eliminare e cercare attività. Questo è un semplice programma di gestione delle attività che consente agli utenti di creare, eliminare, modificare e cercare attività. Ogni attività può essere contrassegnata come completata. Le attività sono archiviate in un file JSON denominato "promemoria.json" nella directory "src". Il programma prevede un sistema di menu basato su testo per interagire con le attività.
